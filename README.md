# 📊 Interactive Power BI Dashboard

## 📌 Overview
This project showcases an interactive **Power BI Dashboard** designed to transform raw data into meaningful insights and support data-driven decision-making.

The focus was on building a **clean, modern, and user-friendly dashboard** that highlights key metrics and trends effectively.

---

## 🎯 Objectives
- Present data in a clear and structured way  
- Improve user interaction and experience  
- Apply time-based analysis for deeper insights  
- Deliver a modern and professional design  

---

## ⚙️ Features
- 📌 Interactive navigation using **Bookmarks**  
- 📊 Time Intelligence analysis (YoY / MoM)  
- 🎯 Clear and focused KPIs  
- 🎨 UI/UX design inspired by Figma  
- 🔍 Dynamic filtering using slicers  
- ⚡ Smooth and user-friendly experience  

---

## 🛠️ Tools & Technologies
- Power BI  
- DAX (Time Intelligence Functions)  
- Figma
