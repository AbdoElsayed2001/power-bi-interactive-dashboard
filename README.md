Power BI Interactive Dashboard
🔹 Overview

This project presents an interactive dashboard built using Power BI to transform raw data into meaningful insights and support data-driven decision making.

The main goal was to create a clean, user-friendly, and visually appealing dashboard that highlights key metrics and trends.

🔹 Features
📌 Interactive dashboard using Bookmarks
📊 Time Intelligence analysis (YoY, MoM comparisons)
🎯 Clear KPI representation
🎨 Professional UI/UX design inspired by Figma
🔍 Dynamic filtering using slicers
🔹 Tools & Technologies
Power BI
DAX (Time Intelligence)
Figma
🔹 Dashboard Preview


🔹 Project Highlights
Focused on improving user experience and interactivity
Designed a clean and modern layout
Published on Power BI Service for accessibility

Download the .pbix file
Open it using Power BI Desktop
Interact with filters and visuals
